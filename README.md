# 🛒 E-Commerce Frontend Project

A fully frontend-based e-commerce web application built using **HTML, CSS, and JavaScript**, designed to simulate a real online shopping experience. This project focuses on UI/UX design, cart logic, navigation flow, and interactive user experience without backend or payment integration.

---

## 🔗 Live Demo

https://dev-rup213.github.io/my-web/index.html

---

## 📌 Project Overview

This project is a **simulated e-commerce platform** that demonstrates how a modern online store works from a frontend perspective. It includes multiple interconnected pages with navigation, product browsing, cart functionality, and user assistance features.

The system is designed to replicate a real shopping flow:

- Users can browse products  
- View product details and descriptions  
- Add items to a cart  
- See total price calculations in real time  
- Navigate easily between pages using a consistent navbar  

No real payment or backend system is implemented; the focus is purely on frontend logic and user experience design.

## 🧭 Website Structure

### 🏠 Home Page
- Landing page of the website  
- Contains “Shop Now” buttons that redirect users directly to the Store page  
- Provides a quick entry point into shopping flow  

---

### 🛍️ Products Page
- Displays all available products  
- Each product includes:
  - Short description  
  - Navigation links to other pages  
- Helps users understand products before adding them to cart  

---

### 🏬 Store Page
This is the core functional area of the project.

Features:
- 🛒 Add to Cart system (simulated)  
- 💰 Real-time total price calculation  
- 📦 Product listing with selection options  
- 🎯 Simple and efficient UI for shopping simulation  

This section mimics a real e-commerce checkout experience.

---

### 🛒 Cart System
- Displays selected products  
- Shows quantity and price breakdown  
- Dynamically updates total cost  
- Designed for clarity and ease of use  

---

### ❓ Help Section
- Contains frequently asked questions  
- Provides guidance on how to use the website  
- Improves user experience for new visitors  

---

### 🔍 Search Feature
- Fully working frontend search system  
- Helps users quickly find products or sections  
- Improves navigation efficiency across the website
